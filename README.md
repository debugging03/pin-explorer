# PIN Explorer 🛡️

A sleek, algorithmic 4-digit PIN combination assistant designed for manual verification sequences.

**[🚀 Click here to use the Web App](https://debugging03.github.io/pin-explorer/)**

## 🎯 What is this for?
Have you ever been locked out of a device (like an old Android phone) where you have to wait minutes between trying different PINs? 

**PIN Explorer** is your assistant for those moments. It doesn't "hack" anything—instead, it helps you systematically try combinations without losing your place. It remembers which numbers you've already tried and suggests the most common patterns first, making a manual process much faster and more organized.

## ✨ Features
- **Smart Logic**: Choose from 5 modes including "Most Frequent" (common PINs like 1234) or "Date-based" (birthdays).
- **Auto-Timer**: Set a countdown that matches your device's lockout period so you know exactly when to try the next code.
- **Progress Tracking**: See exactly how many combinations are left and how much time it might take to finish.
- **History Log**: Keeps a record of failed attempts so you never repeat the same mistake.

## 📦 Local Installation (For Developers)
If you want to run this project on your own computer:

1. **Clone the repository**:
   ```bash
   git clone https://github.com/debugging03/pin-explorer.git
   ```
2. **Setup**:
   ```bash
   npm install
   ```
3. **Launch**:
   ```bash
   npm run dev
   ```

## 🛠️ Tech Stack
- **Core**: React 19 + TypeScript
- **Styling**: Tailwind CSS
- **Motion**: Framer Motion
- **Icons**: Lucide React
- **Build**: Vite
