# PIN Explorer 🛡️

A sleek, algorithmic 4-digit PIN combination assistant designed for manual verification sequences.

## 🚀 How to Use

### Method 1: Use the Web App (Easiest)
Access the live, fully functional tool directly in your browser:
**[https://debugging03.github.io/pin-explorer/](https://debugging03.github.io/pin-explorer/)**

### Method 2: Run Locally (For Developers)
1. **Clone the repo**:
   ```bash
   git clone https://github.com/debugging03/pin-explorer.git
   ```
2. **Install dependencies**:
   ```bash
   npm install
   ```
3. **Start development server**:
   ```bash
   npm run dev
   ```

## ✨ Features
- **5 Algorithmic Modes**: Sequential, Frequency, Date-based, Pattern-based, and Randomized.
- **Custom Timer**: Set switching intervals between 0.5s and 60s for manual verification.
- **Smart Tracking**: Real-time performance stats, estimated completion time, and attempt history.
- **Sleek UI**: Modern, technical interface designed for focus and precision.

## 🛠️ Tech Stack
- **Framework**: React 19 + TypeScript
- **Styling**: Tailwind CSS
- **Animations**: Motion (Framer)
- **Icons**: Lucide React
- **Build Tool**: Vite
